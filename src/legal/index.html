Conditions d'utilisation et traitement des donnees

Informations importantes pour les visiteurs de NinConvert.

NinConvert est un convertisseur audio. En utilisant cet outil, tu acceptes ces conditions.

Donnees traitees

Le backend traite uniquement le fichier audio envoye, le format cible, les options de loop/canaux/frequence et les metadonnees techniques necessaires a la conversion.

Procedure de conversion

Le fichier est recu, normalise en WAV, converti vers le format Nintendo choisi, puis le fichier final est retourne pour telechargement.

Stockage temporaire

Les fichiers intermediaires sont supprimes automatiquement pendant/apres conversion. Le fichier final est conserve temporairement (15 minutes max) puis supprime automatiquement ou via purge manuelle.

Responsabilite et contact

Ne pas envoyer de contenu sensible. L'administrateur peut arreter le backend ou purger les fichiers a tout moment.

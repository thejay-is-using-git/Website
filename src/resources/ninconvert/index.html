VGAudioCli - MIT 
Utilise pour encoder BRSTM / BCSTM / BFSTM.
Les points de loop sont passes via arguments:
-l start-end
FFmpeg (via ffmpeg-static) 
Utilise pour normaliser toutes les sources en WAV 48kHz stereo
avant conversion Nintendo.

Outils locaux pour NinConvert

vgmstream-cli - mixed licenses 
Utilise pour verifier metadata/loop des sorties Nintendo.
Ce n'est pas l'encodeur principal dans ce projet.
ctr_WaveConverter32 - Nintendo SDK tool 
Utilise pour encoder BCWAV.
Loop transmis via:
--loopStart N --loopEnd M
NW4F_WaveConverter - Nintendo SDK tool 
Utilise pour encoder BFWAV.
Loop transmis via:
--loopStart N --loopEnd M
